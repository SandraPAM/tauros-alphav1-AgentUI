# Internal Hackathon Project: Google ADK + Flask

This repository contains a full-stack AI Agent implementation using the Google ADK and a Flask web wrapper.

## Project Structure
* `/data/my_agent`: The core logic for the Google ADK Agent.
* `/data/my_flask_project`: The web interface and API wrapper.
* `setup_test.sh`: Automation script to set up the environment.

## Quick Start
To set up the entire environment automatically, run:
```bash
chmod +x setup_test.sh
./setup_test.sh
